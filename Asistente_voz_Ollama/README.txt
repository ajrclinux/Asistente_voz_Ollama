CODIGO — Instalador para Ubuntu
===============================

Contenidos:
- scripts/install.sh         -> Instalador gráfico (zenity) con asistente
- scripts/run_app.sh         -> Lanzador de la aplicación
- desktop/codigo.desktop     -> Entrada de menú (se instala automáticamente)
- app/app.py                 -> Código de la aplicación
- app/requirements.txt       -> Dependencias Python
- docs/manual_instalacion.md -> Manual de instalación (también PDF si disponible)
- docs/manual_uso.md         -> Manual de uso (también PDF si disponible)

Instalación rápida:
1) Extraiga el .zip
2) En la carpeta extraída, ejecute:
   bash scripts/install.sh
